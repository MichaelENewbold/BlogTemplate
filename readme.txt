For the creation of new blog posts follow these steps.
Steps:
1. Copy the blogpost Template and then rename the file to be somewhat similar to the article name, and does not conflict with other article html names.
2. Go into the copied html and replace the white sections with the intended text.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Steps for Implementation:
1. Save all work.
2. Go to the respective Github, and update the files by dragging in the entire folder of the respective code into a github push.
3. Check Website to make sure the auto grab from render worked fine, and enjoy your fully published website.
